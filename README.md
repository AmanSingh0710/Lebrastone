# 🚀 Lebrostone Project

## 📌 Overview
Lebrostone is a full-stack web application built using modern technologies.  
It provides secure authentication, API integration, and scalable backend services.

---

## 🛠️ Tech Stack
- Frontend: Reactjs , Tailwindcss
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, Google OAuth

---

## ⚙️ Features
- User Authentication (Login/Register)
- Role-based Access Control (Admin/User)
- Secure API Integration
- CRUD Operations
- Responsive UI
